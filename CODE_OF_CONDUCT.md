# 行为准则 / Code of Conduct

## 我们的承诺

为了营造一个开放和友好的社区环境，我们承诺尊重所有参与者——无论其经验水平、性别、性取向、残障、种族、宗教或年龄。

## 准则

有助于创造积极环境的行为：

- 使用友好和包容的语言
- 尊重不同的观点和经历
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员展现同理心

不可接受的行为：

- 使用性别化或其他形式的侮辱性语言或图像
- 人身攻击、政治攻击或无故挑衅
- 公开或私下骚扰
- 未经许可发布他人隐私信息
- 其他在专业环境中被合理认为不恰当的行为

## 执行

社区领导人负责澄清和执行行为准则。对不可接受行为的投诉将得到审查和调查，并将作出被认为适当且符合情况的回应。社区领导人有义务对事件报告人保密。

违反行为准则的后果可能包括从警告到临时或永久封禁。

## 报告

如果你遭受或目睹了违反行为准则的行为，请通过 issue@xtocn.com 联系项目维护者。所有投诉将被审查和及时公正处理。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版。
