## xtcms 是什么

xtcms 是一个开源的轻量级 CMS 框架。不做数据库、不做插件生态——模板即代码，内容即文件，社区共建模板生态。

**核心理念**：一套内容，多种表现。换模板不丢内容，写内容不改代码。

## 技术栈

| 层 | 技术 |
|----|------|
| 服务端渲染 | Astro 6 (SSR mode) |
| 内容管理 | Sveltia CMS（内置） |
| 样式方案 | Tailwind CSS v4 |
| 内容格式 | Markdown + YAML Frontmatter |
| 运行环境 | Node.js 22+ 单进程 |
| 模板语言 | Astro 组件 (`.astro`) |

## 特色

- **模板继承**：子模板只写差异，几十行代码衍生一个全新站点
- **内容与表现分离**：切换到不同模板，内容不变，只改变外貌
- **零数据库**：内容是 `.md` 文件，git 版本管理
- **开箱即用**：一个命令跑起来就是完整网站 + 管理后台
- **多语言**：CMS 后台和前端均支持中英文切换
- **社区模板**：可安装、发布、共享模板
