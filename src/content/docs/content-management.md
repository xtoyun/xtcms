## 访问后台

`http://localhost:4321/admin`，使用 `.env` 中配置的账号登录。

## 内容类型

| 类型 | 说明 | 关键字段 |
|------|------|---------|
| 文章 | 博客文章 | 标题、日期、标签、封面、正文 |
| 页面 | 独立页面 | 标题、封面、正文（自动出现导航） |
| 项目案例 | 客户案例 | 名称、分类、封面、精选 |
| 服务 | 服务介绍 | 名称、封面、描述、排序 |
| SEO关键字 | 自动内链 | 关键字、链接 URL |
| 友情链接 | 侧边栏链接 | 网站名、URL、排序 |
| 留言 | 表单提交 | 姓名、电话、内容 |

## 写文章

1. 左侧菜单 → 文章 → 新建
2. 填写标题、描述
3. 上传封面图（可选）
4. 正文用 Markdown 编辑器
5. 设置标签、草稿状态
6. 保存

## SEO 关键字

关键字会自动在文章正文中匹配生成链接（每篇最多 3 次）。

```
关键字：xtcms → URL：https://xtocn.com
效果：文章中 "xtcms" 自动变链接
```

## 语言切换

CMS 后台根据浏览器语言自动切换中英文。前端同步。
