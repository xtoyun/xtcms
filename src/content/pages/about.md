## 关于本站

这是一个使用 xtcms 1.0 搭建的博客。

xtcms 是一个开源的轻量级内容管理框架，基于 Astro 构建，支持：

- Markdown 内容管理
- 模板继承与定制
- Git 版本控制
- 社区模板生态

### 技术栈

- **前端框架**：Astro 6 (SSR)
- **管理后台**：Sveltia CMS
- **样式**：Tailwind CSS v4 + CSS 变量
- **运行时**：Node.js standalone

### 开源

xtcms 以 MIT 协议开源，欢迎贡献代码和模板。
